## 0. Purpose

This contract specifies the exact conditions under which RT's gravity sector
can claim "validated on cosmic filaments." It is fail-closed: any gate failure
produces an **inconclusive** or **failed** verdict, never a silent pass.

The contract is pre-registered before any holdout data are examined.

---

## 1. Theoretical claim under test

**Claim**: Cosmic filaments are second-Jet anisotropy bridges generated by the
nonlocal closure for kappa*. They are readouts of the STF anisotropy field
Q(x; ell) computed from Liouville moments of the certified closure solution.
The forward model predicts filament transverse shear profiles without
target-specific retuning, using only:
- observed baryonic/galaxy geometry as input,
- the certified nonlocal closure for kappa*,
- the Q(x; ell) readout map,
- a single frozen low-strain anchor mu_IR.

**Null hypothesis**: The observed filament shear signal is fully explained by
first-Jet (bulk/isotropic) content alone, with no second-Jet anisotropy
contribution.

## 2. Primary pre-registered observable

### 2.1 Definition

**Stacked filament transverse shear profile** (SFTSP):

For each filament segment of length L with spine direction hat{e}_parallel:

1. Define transverse coordinate r_perp measured perpendicular to the spine
2. Compute the projected Q-anisotropy component:
   Q_perp(r_perp) := hat{e}_perp^T Q(x; ell) hat{e}_perp
   evaluated on the transverse slice
3. Stack (average) Q_perp(r_perp) across all filaments in the roster
   using inverse-variance weighting
4. The primary statistic is the **integrated transverse anisotropy amplitude**:
   A_fil := integral from 0 to R_max of Q_perp(r_perp) * r_perp dr_perp

where R_max is the certified corridor half-width (frozen before holdout evaluation).

### 2.2 Observational counterpart

The weak-lensing-derived counterpart is the **stacked filament tangential
shear profile** gamma_t(r_perp), which is the standard observable in filament
weak-lensing studies. The mapping between Q_perp and gamma_t is:

gamma_t(r_perp) = (b / c^2) * integral[Q_perp * Sigma_cr^{-1}] along LOS

with b the strain-optic coefficient from Axiom 1 eq. (3), and Sigma_cr the
critical surface density from the lens-source geometry.

### 2.3 Secondary observables (exploratory, not gate-bearing)

- Principal-direction coherence along spine
- Bridge connectivity graph topology
- Filament length-integrated anisotropy norm ||Q||_F
- Longitudinal vs transverse anisotropy ratio

These are reported but do NOT enter pass/fail adjudication.

## 4. Certified scale and smoothing

### 4.1 Readout scale ell

ell = 2 Mpc/h (comoving)

**Justification**: This is the characteristic transverse scale of cosmic
filaments in the literature (typical filament diameter 2-5 Mpc/h). It sits
well within the regime where the nonlocal closure is expected to produce
measurable anisotropy bridges.

### 4.2 Corridor half-width R_max

R_max = 5 Mpc/h (comoving, transverse to spine)

### 4.3 Smoothing kernel

Gaussian kernel with sigma = ell / sqrt(2) = 1.414 Mpc/h

### 4.4 Freeze rule

ell, R_max, and the kernel are frozen at contract signing. No post-hoc
adjustment is permitted. If scale choice proves unsuitable, the verdict is
**inconclusive**, not "we need a different scale."

## 5. Filament metrology roster

### 5.1 Training set (mu_IR anchor): 3 filaments

Selection criteria (applied before any RT computation):
- Length >= 15 Mpc/h
- At least 2 spectroscopic galaxy clusters (richness >= 20) at endpoints
- KiDS-1000 weak-lensing coverage with source density >= 5 arcmin^{-2}
- Redshift range 0.1 <= z <= 0.3
- No known strong lensing contamination
- No masking artifacts covering > 20% of corridor footprint

The 3 highest-signal filaments satisfying all criteria are selected, ranked by
endpoint cluster richness sum. Selection is performed ONCE and hashed.

### 5.2 Holdout set: 8 filaments

The next 8 filaments satisfying the same criteria, in the same ranking.

### 5.3 Freeze discipline

- Training roster is frozen before any mu_IR estimation.
- Holdout roster is frozen before any holdout evaluation.
- Both rosters are hashed (SHA-256) at freeze time.
- No filament may move between training and holdout.
- No additional filaments may be added post-freeze.

## 6. Low-strain anchor: mu_IR

### 6.1 Definition

mu_IR is the IR shear modulus of the MoP in the low-strain (gravity) regime.
It enters the nonlocal closure functional as the shear penalty coefficient:

E_ell[kappa] = integral[ (1/2) lambda_IR theta^2
                         + mu_IR <epsilon^dev, epsilon^dev>
                         + alpha |nabla epsilon|^2 ] d^d x
               + boundary/constraint terms

### 6.2 Anchoring protocol

1. For each training filament:
   a. Ingest baryonic geometry (cluster positions, masses from richness-mass relation)
   b. Solve closure for kappa* as function of mu_IR (parametric family)
   c. Compute Q(x; ell) and derive predicted SFTSP
   d. Compare to observed stacked gamma_t profile from KiDS-1000

2. Determine mu_IR by minimizing the chi-squared between predicted and
   observed SFTSP over the 3 training filaments simultaneously:
   mu_IR* = argmin_{mu_IR > 0} sum_{i=1}^{3} chi^2_i(mu_IR)

3. Report: value, uncertainty, residual budget, condition number.

### 6.3 Estimand Stability Gate (ESG)

Before freezing mu_IR*, the following must pass:
- Leave-one-out stability: removing any single training filament changes
  mu_IR* by less than 15%
- Solver convergence: residual < 10^{-6} for each training filament
- Condition number of the Fisher matrix < 100

If ESG fails: verdict = **inconclusive**, no holdout testing proceeds.

### 6.4 Freeze certificate

Once mu_IR* passes ESG:
- Value is frozen with full provenance chain
- SHA-256 hash of (training data, solver version, mu_IR* value)
- No further modification permitted

## 7. Forward model protocol (holdout evaluation)

For each holdout filament (i = 1, ..., 8):

1. **Ingest**: baryonic geometry from SDSS (galaxy/cluster positions, redshifts)
2. **Solve**: nonlocal closure for kappa*_i using frozen mu_IR*, lambda_IR, alpha
3. **Compute**: Q_i(x; ell) from Liouville moments
4. **Derive**: predicted SFTSP_i = Q_perp,i(r_perp), integrated amplitude A_fil,i
5. **Compare**: to observed stacked gamma_t,i from KiDS-1000

**Forbidden**: Any refit of mu_IR, lambda_IR, alpha, ell, R_max, kernel, or
any other parameter between holdout filaments.

---

## 8. Pass / fail / inconclusive criteria

### 8.1 Primary gate: holdout prediction accuracy

**PASS** if:
- For >= 6 of 8 holdout filaments, the predicted A_fil,i is within
  the 2-sigma observational uncertainty of the measured value
- The stacked holdout residual (predicted - observed) is consistent with
  zero: |mean residual| < 2 * sigma_{stacked}
- The reduced chi-squared over all holdout filaments satisfies:
  0.25 < chi^2_red < 4.0

**FAIL** if:
- Fewer than 4 of 8 holdout filaments are within 2-sigma
- OR the stacked residual exceeds 5-sigma
- OR chi^2_red > 10 (systematic model failure)
- OR chi^2_red < 0.1 (suspiciously good — indicates overfitting leak)

**INCONCLUSIVE** if:
- Between the PASS and FAIL thresholds
- OR any upstream gate (ESG, data quality, systematics) fails
- OR observational uncertainties are too large to distinguish RT from null

### 8.2 Secondary diagnostic (non-gate-bearing)

- Profile shape comparison (not just amplitude)
- Filament-by-filament scatter vs. expected cosmic variance
- Consistency between KiDS and DES cross-check

## 9. Null tests (mandatory)

All null tests must be executed on the holdout set with frozen parameters.

### 9.1 First-Jet null (lambda_2 = 0)

Force the second-Jet content to zero (set the deviatoric/anisotropy channel
to zero in the closure). The predicted SFTSP should collapse to zero (or to
the isotropic background level).

**Gate**: |A_fil^{null,1J}| < 0.1 * |A_fil^{full}| for >= 6 of 8 holdout filaments.
**Failure mode**: If the first-Jet null still produces significant filament
signal, the theory's second-Jet necessity claim is falsified.

### 9.2 Isotropy null

Replace Q(x; ell) with its isotropic average (1/d) tr(Q) * I.
The directed filament response should vanish.

**Gate**: |A_fil^{null,iso}| < 0.1 * |A_fil^{full}| for >= 6 of 8 holdouts.

### 9.3 Scrambled geometry null

Randomize source-pair associations: assign each filament's endpoint clusters
to random positions drawn from the cluster catalog (preserving redshift
distribution). Bridge coherence should vanish.

**Gate**: |A_fil^{null,scram}| < 0.15 * |A_fil^{full}| (stacked over all holdouts).

### 9.4 Scale robustness test

Recompute with ell' = 0.7 * ell and ell'' = 1.4 * ell.
The signal should persist (within a factor of 2) and not change sign.

**Gate**: 0.3 < A_fil(ell') / A_fil(ell) < 3.0 AND same sign, for stacked holdout.

### 9.5 Survey systematics test

Correlate the filament-by-filament residual (predicted - observed) with:
- Survey depth at filament location
- PSF size at filament location
- Source galaxy density
- Mask fraction in corridor

**Gate**: |Pearson r| < 0.5 for each correlation. Any |r| > 0.7 triggers
FAIL (systematics-dominated).

## 10. What counts as "validated"

RT can claim **"validated on cosmic filaments"** if and only if ALL of:

1. The filament targets were public and pre-registered (this contract)
2. The low-strain anchor mu_IR was frozen before holdout evaluation (Sec 6)
3. The forward model predicted holdout observables with no refit (Sec 7)
4. The primary gate PASSED (Sec 8.1)
5. ALL five null tests passed their gates (Sec 9)
6. The first-Jet null failed exactly as theory predicts (signal collapse)

If any one condition is not met:
- **Exploratory**: if nulls not yet run
- **Provisional**: if primary gate passed but some nulls not yet run
- **Inconclusive**: if gates are ambiguous or data quality insufficient
- **Failed**: if primary gate or critical null test failed
- **Falsified (second-Jet necessity)**: if first-Jet null does NOT collapse

## 11. Residual budget

Each step in the pipeline reports:

| Step | Residual type | Tolerance |
|------|--------------|-----------|
| Closure solve | Solver convergence | < 10^{-6} |
| Log-strain | Linearization error | < 10^{-4} relative |
| Q readout | Discretization | < 5% of signal |
| Lensing projection | LOS integration | < 2% of signal |
| Stacking | Bootstrap variance | Reported, not gated |

Total residual budget: quadrature sum < 10% of predicted signal amplitude.
If exceeded: verdict = **inconclusive**.
